# Edit-photo-date-from-csv-files
Small pyhton script to help input dates into photes missing DateTimeOriginal in the EXIF metadata


I had a problem with photes downloaded from Icloud, Alot of files did not have a date in the metadata.
Alot of files had the same names. The icloud servers are case sensitive, but the exfat external hdd i was using is not. So the script renames all files to the DateTimeOrginal
